Participants are required to play the 4 tables labelled, (table1a – table 2a) and (table 1b – table 2b). Each table has 6 decisions and combined make 24 decisions. Each table has an Option A (Lottery value package) and an Option B (Sure value). Compare the Lottery value package (Option A) with each value of the Sure value in (Option B). This means that in TABLE 1a for example, looking at Option B there are 6 Sure Values that are guaranteed incentives (R50, R60, R70, R80, R90 and R100). In column Option B, compare EACH Sure Value with the Lottery Value Package in Option A (R120 with 25% probability and R40 with 75% probability). For example; compare the R50 in Option B with the R120 with 25% probability & R40 with 75% probability in Option A. Would you rather get the R50 or get ONE of the Lottery Value in Option A. Make your decision in 'Decision 1'. If you choose the sure value of R50, tick 'Option B' in Decision 1. If you choose the lottery value package (R120 with 25% probability and R40 with 75% probability), tick 'Option A' in Decision 1. Do this in every decision. Tick a box to make your preferred choice of either Option A (Lottery value package) or Option B (Sure value). Do this in all 4 tables.
